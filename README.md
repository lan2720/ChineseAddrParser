
# ChineseAddrParser

A Chinese Address Parser written by Python


### Usage

 1. `git clone git@github.com:lan2720/ChineseAddrParser.git`
 2. cd the directory and run `python addrParse.py 上海近郊青浦区沪青平公路2933弄`
